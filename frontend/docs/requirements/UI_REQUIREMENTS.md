# 보아스 UI 화면 설계 요구사항
## 화면 구성 및 기능 명세

이 문서는 보아스 측정 데이터 관리 시스템의 모든 화면 구성과 기능을 정의합니다.

---

## 전체 레이아웃 구조

### 기본 레이아웃
```
┌─────────────────────────────────────────────────────────┐
│  [로고] 보아스 환경측정     [대시보드][측정][고객][리포트]│  ← 상단 네비게이션
├──────────┬─────────────────────────────────────────────┤
│ 📊 대시보드 │                                            │
│ 📝 측정입력 │                                            │
│ 📋 측정이력 │          메인 컨텐츠 영역                    │
│ 📈 통계차트 │                                            │
│ 🏢 고객사   │                                            │
│ 🏭 굴뚝관리 │                                            │
│ 📄 리포트   │                                            │
│ ⚙️ 설정     │                                            │
└──────────┴─────────────────────────────────────────────┘
```

### 레이아웃 사양
- **상단바 높이**: 64px
- **사이드바 너비**: 240px (접었을 때 64px)
- **메인 영역**: 나머지 공간 전체
- **반응형**: 태블릿/모바일에서 사이드바 오버레이

### 색상 테마
- **주 색상**: 파란색 계열 (#1E40AF, #3B82F6, #60A5FA)
- **배경색**: 밝은 회색 (#F9FAFB, #F3F4F6)
- **텍스트**: 진한 회색 (#111827, #374151)
- **성공**: 초록색 (#10B981)
- **경고**: 주황색 (#F59E0B)
- **위험**: 빨간색 (#EF4444)

## 1. 기본 레이아웃 (Layout)

### 1-1. 상단 네비게이션 바

**구성 요소:**
```
[로고/회사명] [빈공간] [대시보드][측정관리][고객사관리][리포트] [알림🔔] [프로필👤]
```

**기능:**
- 로고 클릭 → 홈 대시보드
- 메뉴 항목 hover → 하위 메뉴 드롭다운
- 알림 아이콘 → 기준 초과 알림 팝업
- 프로필 → 내 정보, 로그아웃

### 1-2. 사이드바 메뉴

**메뉴 구조:**
```
📊 대시보드
   └─ 홈 대시보드
   └─ 통계 차트

📝 측정 관리
   └─ 측정 데이터 입력
   └─ 측정 이력
   └─ 기준 초과 이력

🏢 마스터 관리
   └─ 고객사 관리
   └─ 굴뚝 관리
   └─ 배출허용기준 설정
   └─ 측정항목 관리
   └─ 측정업체 관리

📄 리포트
   └─ 측정기록부
   └─ 월간 보고서
   └─ 기간별 보고서

⚙️ 설정
   └─ 사용자 관리
   └─ 시스템 설정
```

**인터랙션:**
- 메뉴 아이콘 클릭 → 사이드바 접기/펼치기
- 현재 페이지 강조 표시
- 하위 메뉴 확장/축소

## 2. 측정 데이터 입력 화면 ⭐⭐⭐

### 2-1. 화면 구조

```
┌─────────────────────────────────────────────────┐
│ 측정 데이터 입력                    [임시저장] [저장]│
├─────────────────────────────────────────────────┤
│ [1단계: 기본 정보]                               │
│ ┌──────────────────────────────────────────┐   │
│ │ 측정일시: [2025-10-26] [14:30]          │   │
│ │ 고객사:   [A회사 ▼]                     │   │
│ │ 굴뚝:     [B-901 ▼]                     │   │
│ │ 측정업체: [보아스환경기술 ▼]             │   │
│ └──────────────────────────────────────────┘   │
│                                                  │
│ [2단계: 측정항목] ────────────────────────      │
│ 탭: [먼지·THC] [중금속] [산화물] [BTEX] [염소계] [기타]│
│                                                  │
│ ┌─────────────────────────────────────────┐   │
│ │ 항목명      | 측정값 | 단위  | 기준  | 상태 │   │
│ ├─────────────────────────────────────────┤   │
│ │ 먼지        | [23.5] | mg/S㎥ | 30   | ✓  │   │
│ │ THC        | [12.3] | ppm   | 50   | ✓  │   │
│ │ 카드뮴(Cd)  | [불검출]| mg/S㎥ | 1.0  | ✓  │   │
│ │ ...                                      │   │
│ └─────────────────────────────────────────┘   │
│                                                  │
│ [3단계: 기상 데이터]                             │
│ ┌──────────────────────────────────────────┐   │
│ │ 풍향: [북풍 ▼]  풍속: [2.3] m/s         │   │
│ │ 기온: [15.2] ℃  기압: [1013] hPa       │   │
│ └──────────────────────────────────────────┘   │
│                                                  │
│ [4단계: 배출가스 특성]                           │
│ ┌──────────────────────────────────────────┐   │
│ │ 유량: [1234] S㎥/min  온도: [120] ℃    │   │
│ │ 산소: [8.5] %        수분: [12.3] %    │   │
│ └──────────────────────────────────────────┘   │
│                                                  │
│ [5단계: 요약 및 검증]                            │
│ ┌──────────────────────────────────────────┐   │
│ │ ✓ 39개 항목 중 37개 입력 완료              │   │
│ │ ⚠️ 먼지 농도가 기준의 78% (주의)           │   │
│ │ ✓ 기준 초과 항목 없음                      │   │
│ └──────────────────────────────────────────┘   │
│                                                  │
│                    [취소] [임시저장] [저장하기]    │
└─────────────────────────────────────────────────┘
```

### 2-2. 핵심 기능

**입력 보조:**
- "불검출" 일괄 체크 버튼
- 이전 측정값 불러오기
- 엑셀 데이터 붙여넣기

**실시간 검증:**
- 숫자 형식 자동 체크
- 배출허용기준 초과 즉시 표시
- 필수 항목 미입력 경고

**상태 표시:**
- 정상: 초록색 체크 ✓
- 주의 (80% 이상): 주황색 ⚠️
- 초과: 빨간색 ❌

### 2-3. 카테고리별 탭 구성

**탭 1: 먼지·THC (2개 항목)**
- 먼지
- 탄화수소(THC)

**탭 2: 중금속 (8개 항목)**
- 카드뮴, 구리, 납, 니켈, 아연, 수은, 크롬, 비소

**탭 3: 산화물·가스 (9개 항목)**
- SOx, NOx, CO, HF, Cl₂, HCl, HCN, H₂S, NH₃

**탭 4: BTEX (4개 항목)**
- 벤젠, 톨루엔, 에틸벤젠, 자일렌

**탭 5: 염소계 화합물 (8개 항목)**
- 디클로로메탄, 클로로포름, 사염화탄소 등

**탭 6: 기타 유기화합물 (8개 항목)**
- 아크릴로니트릴, 페놀, 포름알데히드 등

- 키워드 검색 (고객사명, 굴뚝명)

**정렬:**
- 측정일시 (기본: 최신순)

- 초과 건수

**액션:**
- [보기] → 측정 상세 화면

**일괄 처리:**

- Excel 다운로드

## 공통 UI 컴포넌트

### 버튼
- **Primary**: 파란색, 주요 액션
- **Secondary**: 회색, 보조 액션
- **Danger**: 빨간색, 삭제/위험 액션
- **크기**: Large (48px), Medium (40px), Small (32px)

### 입력 필드
- **Text Input**: 단일 줄 텍스트
- **Number Input**: 숫자 전용 (스피너 포함)
- **Select Dropdown**: 선택 드롭다운
- **Date Picker**: 날짜 선택
- **Checkbox**: 체크박스
- **Radio**: 라디오 버튼

### 알림/상태
- **Success Toast**: 초록색, 성공 메시지
- **Error Toast**: 빨간색, 오류 메시지
- **Warning Toast**: 주황색, 경고 메시지
- **Info Toast**: 파란색, 정보 메시지

### 테이블
- 정렬 가능한 헤더
- 페이지네이션
- 행 선택 (체크박스)
- 반응형 (모바일에서 카드형)

### 모달
- 확인 모달
- 폼 입력 모달
- 이미지/차트 확대 모달
