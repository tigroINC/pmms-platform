## ES 표준 측정방법

### ES 코드 체계
**ES XXXXX.Xy**
- ES: 환경오염공정시험기준 (Environmental Standard)
- XXXXX: 항목 코드
- X: 버전 (1=초판, 2=개정1판...)
- y: 세부 개정 (a, b, c, d...)

### 주요 ES 방법군

| ES 코드 범위 | 대상 항목군 | 측정 방법 |
|-------------|------------|----------|
| ES 01301-01310 | 먼지, 중금속 | 흡광광도법, 원자흡수분광법 |
| ES 01401-01409 | 산화물, 산성가스 | 화학적 흡수법, 기기분석법 |
| ES 01501-01521 | 유기화합물 | 가스크로마토그래피 |

## 단위 정보

### 주요 단위
- **ppm**: parts per million (백만분율)
- **mg/S㎥**: 밀리그램/표준세제곱미터
- **ppb**: parts per billion (10억분율) - 일부 항목

### 단위 변환
- 1 ppm = 1,000 ppb
- ppm ↔ mg/S㎥ 변환은 분자량과 온도/압력에 따라 다름

## 특수 처리 사항

### 1. "불검출" 처리
**입력 가능 값:**
- "불검출"
- "ND" (Not Detected)
- "N.D."
- "<검출한계"

**저장 방식:**
- 데이터베이스: NULL 또는 0
- 화면 표시: "불검출"
- 통계 계산: 0으로 처리 또는 제외

### 2. 검출한계값
각 항목마다 검출한계가 다름
- 예: 수은(Hg) - 0.001 mg/S㎥
- 예: 카드뮴(Cd) - 0.01 mg/S㎥

### 3. 배출허용기준 초과 체크
**기준 적용:**
- 일반지역 기준
- 특별대책지역 기준 (더 엄격)
- 굴뚝별 개별 기준 설정 가능

**표시 방식:**
- 정상: 검은색
- 주의 (80% 이상): 주황색
- 초과: 빨간색

### 4. 표준산소농도 보정
**적용 대상:** 
- 연소시설에서 배출되는 가스
- SOx, NOx, 먼지 등

**보정 공식:**
```
보정농도 = 측정농도 × (21 - 표준산소농도) / (21 - 실측산소농도)
```

**표준산소농도:**
- 일반 연소시설: 12%
- 가스 연소시설: 3%
- 액체 연료: 4%

- "불검출" 일괄 체크 버튼

- 엑셀 복사-붙여넣기 지원

### 4. 실시간 검증
- 숫자 형식 체크
- 범위 검증 (0~999999)
- 배출허용기준 초과 즉시 알림

## 데이터베이스 스키마 권장

```sql
CREATE TABLE measurement_items (
  id SERIAL PRIMARY KEY,
  code VARCHAR(10) UNIQUE,        -- 'ITEM001' ~ 'ITEM039'
  name_ko VARCHAR(100),            -- '먼지'
  name_en VARCHAR(100),            -- 'Particulate Matter'
  category VARCHAR(50),            -- 'DUST', 'HEAVY_METAL', etc
  unit VARCHAR(20),                -- 'mg/S㎥', 'ppm'
  es_code VARCHAR(20),             -- 'ES 01301.1e'
  detection_limit DECIMAL(10,6),   -- 검출한계
  default_limit DECIMAL(10,3),     -- 기본 배출허용기준
  sort_order INT                   -- 표시 순서
);

CREATE TABLE measurement_details (
  id SERIAL PRIMARY KEY,
  measurement_id INT,              -- 측정 메인 ID
  item_id INT,                     -- 측정항목 ID
  measured_value DECIMAL(10,3),    -- 측정값
  is_not_detected BOOLEAN,         -- 불검출 여부
  corrected_value DECIMAL(10,3),   -- 보정값
  exceeds_limit BOOLEAN,           -- 기준 초과 여부
  remarks TEXT                     -- 비고
);
```

## 통계 분석 시 주의사항

### 1. 불검출 데이터 처리
- **옵션 A**: 0으로 계산
- **옵션 B**: 통계에서 제외
- **옵션 C**: 검출한계값의 1/2로 계산

### 2. 이상치 처리
- 과도하게 높은 값 (>10배 평균)
- 물리적으로 불가능한 값

### 3. 트렌드 분석
- 월별 평균, 최대, 최소
